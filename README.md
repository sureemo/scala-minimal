# scala-minimal

## Software Requirements

Scala 2.11.8
http://scala-lang.org/download/

SBT (Scala Build Tool)
http://www.scala-sbt.org/download.html

## How to run

```sh
$ sbt run
```

#### Expected Result

> Hello World

## How to build package

```sh
$ sbt package
```

#### Expected Result

New build file generated at:

> target\scala-2.11\hello_scala_2.11-1.0.jar
